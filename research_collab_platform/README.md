# Research Collaboration Platform
## Overview

The Research Collaboration Platform is a web-based application designed to connect researchers, students, and academic teams. It provides a centralized environment for sharing research work, exploring collaboration opportunities, and managing academic interactions efficiently.

The platform helps users discover research teams, upload and analyze CVs, and engage in meaningful academic collaboration.

## Features
User authentication and profile management
CV upload and analysis
Exploration of research teams and members
Access to collaboration and project opportunities
Responsive and modern user interface
Backend API for data processing and integration
## Technologies Used
Frontend
React
Tailwind CSS
Backend
Flask (Python)
## Tools
Git and GitHub for version control
## Installation and Setup
1. Clone the repository
git clone https://github.com/Nada-belmiloud/research_collab_platform.git
cd research_collab_platform
2. Backend setup
cd backend
pip install -r requirements.txt
python app.py
3. Frontend setup
cd frontend
npm install
npm run dev
## Usage
Create an account and log in
Upload your CV
Explore research teams and available opportunities
Connect with other users and collaborate
